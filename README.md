### Fast-Tool

some tool collection.

#### Install
```sh
npm install fast-tool -g
```

#### Example
```sh
> ftool md5 123456
or
> ftool m 123456

output:
  Input:      123456
  Md5:        e10adc3949ba59abbe56e057f20f883e
  Md5(upper): E10ADC3949BA59ABBE56E057F20F883E
```

#### Support
- md5
- time format convert
- base64
